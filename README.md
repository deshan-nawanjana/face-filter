# A Face Filter App With TensorFlow and Three.js

Live App: [https://deshan.lk/apps/face-filter](https://deshan.lk/apps/face-filter)

Description: [https://deshan.lk/projects/face-filter](https://deshan.lk/projects/face-filter)

Blog: [https://deshan.lk/blog/face-filter-app-with-tensorflow-and-three-js](https://deshan.lk/blog/face-filter-app-with-tensorflow-and-three-js)

Face filter apps are very popular on social media platforms. The basic functionality of a face filter app is to detect faces from a live camera feed, track facial landmarks, and align a 2D or 3D model, such as sunglasses, masks, or animal ears, to the face in real time, creating the effect that the user is wearing those virtual elements.
